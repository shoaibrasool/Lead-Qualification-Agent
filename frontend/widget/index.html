Lead Qualification Widget

Test page for the floating chat widget. Click the bubble in the bottom-right corner to start a conversation.

Using the widget

  1. Click the chat bubble (bottom-right)
  2. Type a message and press Enter or click the send button
  3. Watch for the typing indicator while the bot responds
  4. If time slots appear, click one to select it

API endpoint

The widget sends POST requests to /chat with:

{
  "session_id": "<auto-generated>",
  "message": "your message"
}
